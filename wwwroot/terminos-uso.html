📋 Términos de Uso

Aceptación de términos

Al acceder y utilizar el Sistema SWGROI, usted acepta cumplir con estos términos de uso.

Uso autorizado

El sistema está destinado exclusivamente para:

  • Gestión de tickets de soporte
  • Comunicaciones internas
  • Manejo de documentos corporativos
  • Generación de reportes autorizados

Prohibiciones

Queda estrictamente prohibido:

  • Compartir credenciales de acceso
  • Intentar acceder a información no autorizada
  • Modificar o alterar el sistema
  • Usar el sistema para fines no corporativos

Responsabilidades del usuario

  • Mantener la confidencialidad de sus credenciales
  • Reportar cualquier incidente de seguridad
  • Usar el sistema de manera responsable
  • Cumplir con las políticas corporativas

Limitación de responsabilidad

El sistema se proporciona "tal como está" y el usuario asume toda responsabilidad por su uso adecuado.

Vigencia

Estos términos permanecen vigentes mientras mantenga acceso al sistema.
